# AI Chat Desktop App - Build Workflow

## Architecture
- **Framework**: Tauri v2 (Rust backend + Web frontend)
- **Frontend**: React + TypeScript + Tailwind CSS
- **AI Backend**: Claude CLI subprocess integration
- **UI**: Claude-like chat interface with markdown support

## Features
- Chat conversation with streaming responses
- Markdown rendering (code blocks, tables, etc.)
- Conversation history management
- Multiple conversation tabs
- Dark/light theme
- System prompt configuration
